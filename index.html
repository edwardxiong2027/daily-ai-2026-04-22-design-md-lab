Design MD Lab

A browser-native playground inspired by Google Stitch's newly open-sourced DESIGN.md format. Write an agent-readable design spec, lint it live, preview the resulting UI direction, and run instant accessibility checks before handing it to a model or coding agent.

Live lintingFind missing sections, weak token coverage, unclear goals, and absent accessibility guidance.
Instant previewTurn design tokens and product intent into a visual mockup without leaving the page.
WCAG auditCheck contrast ratios on core token pairs so agents do not hallucinate inaccessible palettes.
Why this exists now Google opened up the draft DESIGN.md spec so AI systems can reason about design intent instead of guessing from screenshots. This app turns that idea into a practical, shippable workflow.
  • New format: DESIGN.md is positioned as a shared visual language across tools and agents.
  • New workflow: Autonomous research and coding agents are expanding fast, so structured handoff artifacts are suddenly valuable.
  • New pain point: Teams need human-readable + model-readable design prompts with built-in accessibility safeguards.
